Theme Toggle Demo

Welcome to the Theme Toggle Component

This component uses pure HTML, Tailwind CSS, and Alpine.js to create a smooth theme switching experience.

  • Remembers your preference in localStorage
  • Respects system preference on first visit
  • Smooth animations and transitions
  • Fully accessible with ARIA attributes

Usage Example 

<div x-data="themeToggle">
    <button @click="toggle()">
        Toggle Theme
    </button>
</div>
